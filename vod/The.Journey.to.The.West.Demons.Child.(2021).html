The.Journey.to.The.West.Demons.Child.(2021)

kilasan singkat

Lima ratus tahun yang lalu Raja Kera membuat kerusuhan di Istana Surgawi, tanpa sengaja menendang kompor langit dan menyebabkan Api Surgawi jatuh ke dunia. Untuk menyelamatkan rakyat jelata, Putri Kipas Besi mencoba yang terbaik untuk menekan api, tetapi Bocah Merah di perutnya terkikis oleh api asli samida dan menjadi bocah iblis.
